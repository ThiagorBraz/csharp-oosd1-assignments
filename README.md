# C# Object-Oriented Software Development 1 — Assignments

Assignments developed for the Object-Oriented Software Development 1 
module at Atlantic Technological University (ATU), Sligo, Ireland.

## Assignments

### CA1 — OOP Fundamentals
Five exercises covering core C# and object-oriented programming 
concepts including classes, methods, loops and conditionals.

### CA 2
C# programming assignment focusing on object-oriented design 
and implementation.

### CA3 — BlackJack Game
A fully playable console-based BlackJack card game built in C#, 
applying object-oriented principles such as classes, inheritance 
and encapsulation.

## Technologies Used

- C# / .NET
- Visual Studio 2022
- Object-Oriented Programming principles

## Academic Context

Developed at Atlantic Technological University (ATU), Sligo — 
Higher Diploma in Computer Science, 1st Semester, OOSD 1 Module.
