# C# Object-Oriented Software Development 1 — Assignments

Assignments developed for the Object-Oriented Software Development 1 
module at Atlantic Technological University (ATU), Sligo, Ireland.

## Assignments

### CA1 — OOP Fundamentals
Five console-based exercises covering core C# and object-oriented 
programming concepts including:
- Classes and objects
- Methods and constructors
- Loops and conditionals
- Collections and data types

### Assessment 2 — Football League Table
A console-based application for managing a football league table. 
Features include:
- Input team data: name, wins, draws, losses, goals for 
  and goals against
- Store multiple teams using a List collection
- Display and sort teams by performance
- Built with OOP principles including classes, static methods 
  and collections

### CA3 — BlackJack Card Game
A fully playable console-based BlackJack card game built in C#. 
Features include:
- Full deck of cards represented as objects
- Player and Dealer classes
- Game logic with Hit and Stand mechanics
- Hand evaluation and winner determination
- Built with OOP principles including classes, collections, 
  encapsulation and game logic

## Technologies Used

- C# / .NET

- Console applications
- Object-Oriented Programming principles
